# Part 9: Data Governance
Data governance has many aspects to it. But we are limiting the scope to access management and data lineage in this part. 



## Outline

## Access control for tables and views
1. Based on your data model, define access rules to various tables in dev and prod to groups like 'ANALYSTS', 'DATA_ENGINEERS' and 'BUSINESS_USER'.
1. Implement access control to tables. Please make sure to have below scenarios in access control
    - READ ONLY access
    - COMPLETE access
    - Row level security
    - Column masking

**Deliverables**

1. Access control rules defined and implemented in dev and prod.

## Data Lineage

1. Use Unity catalogue's capabilities to genrate data lineage information on your gold layer tables.


**Deliverables**

1. Data lineage available in Unity Catalogue for gold layer tables.
